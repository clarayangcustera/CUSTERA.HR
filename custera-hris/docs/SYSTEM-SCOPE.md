# Custera HRIS - Functional scope

This original system was structured from publicly documented HRMS workflows and the submitted reference screens. It is deliberately not a visual or code copy of any third-party product.

## Included and functional

1. Authentication and role-based access
2. Employee master data
3. Departments and organisation reference data
4. Leave types, annual leave entitlements, application and approval
5. Claim types, expense claim submission and approval
6. Daily attendance clock-in / clock-out
7. Draft payroll runs, payslips and publication state
8. Document register with organisation / employee visibility
9. Audit log

## Recommended next modules

1. Password reset, invitation email and multi-factor authentication
2. File upload API with object storage, anti-malware scan and signed URLs
3. Generic multi-step approval workflow engine
4. Public-holiday, roster, overtime, late-arrival and field check-in policies
5. Employee onboarding/offboarding checklists
6. Recruitment, job requisition and candidate portal
7. Singapore / Malaysia statutory payroll calculator isolated behind a versioned tax-rule adapter
8. Notifications and report exports
9. Support for multiple legal entities and cross-organisation web accounts
10. Security and privacy test review before storing real payroll or personal data
